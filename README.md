# pipi-chan [WIP]

EDTCのアイデアコンテストにおける二年生の共同プロジェクト

## プロジェクトの意義、目的

- EDTCには今までない「ゲーム」という新規性
- ソフトウェア開発という技術的新規性
- S科では得られないIoT開発による挑戦

このプロジェクトの目的は、子供たちにものづくりの魅力を伝えることが最終的な目標ではあるが、
このプロジェクトの開発が今後のEDTCのIoT開発の糧になることも期待している。

## 主な機能

- ユニークなバーコードを持ったカードをセンサによって読み込む
- 読み込んだカード情報をWebサーバで加工してキャラクターとして表示する

## 技術スタック

キャラクターを表示するWebサイトは別途用意することを予定。  
Webサイトにはキャラクターの表示以外の機能を追加する予定なので、複雑なロジックは組み込みのプログラムに詰め込みたくない。  
仮に、Webサイト以外のソフトウェアを用いる場合でも同じ思想。

## 環境構築

> [!IMPORTANT]
> このプロジェクトはVSCodeで開発を行うことを想定している。

### PlatformIO IDE 設定手順

ESP32と通信をするための拡張機能を入れる。  
以下のリンクまたは、VSCodeの拡張機能タブで「PlatformIO IDE」と検索する。  
[PlatformIO IDE](https://marketplace.visualstudio.com/items?itemName=platformio.platformio-ide)  

この時に、左のタブにハチ？のようなタブが追加されていることを確認する。  
これがPlatformIOになる。

次に、このリポジトリをクローンする。  
以下のコマンドを実行したディレクトリに**pipi-chan**がクローンされる。

```bash
git clone https://github.com/fami-gb/pipi-chan.git
```

次に、左のタブからPlatformIOを開き、クローンしたディレクトリを開く。

以上で環境構築は終了。

### ESP32へプログラムをアップロードする。

ショートカットを使うのが手っ取り早い。  
ESP32とUSB接続しているなら`Ctrl + Alt + U`でビルドとアップロードを同時に行える。

そうでなくても、VSCode上部にある「✓」の右にある逆三角から`Upload`を選ぶと同様のことを行える。
